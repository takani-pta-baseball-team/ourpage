📖 集計ルール / 自動判定について

このサイトでは、各試合の打席記録から成績やスコアを自動集計しています。 判定ルールは以下のとおりです。

目次

  1. 打席結果の種類
  2. 打撃成績の自動集計
  3. 投手成績の自動集計
  4. 勝ち / 負け投手の自動判定
  5. イニングスコア・試合結果
  6. イニングの自動進行
  7. MVP・その他

1. 打席結果の種類

各打席で1つの結果を記録します。全12種類を3つのグループに整理しています。

ヒット系

  • ヒット: 単打
  • 二塁打: 2塁まで進塁
  • 三塁打: 3塁まで進塁
  • 本塁打: ホームラン

出塁系（アウトじゃないけどヒットでもない）

  • 四球: ボール4つでフォアボール（打数に含まない）
  • 死球: ボールがバッターに当たって出塁（打数に含まない）
  • エラー: 失策で出塁
  • 他セーフ: その他の理由で出塁（フィルダースチョイスなど）

アウト系

  • 三振: ストライクアウト
  • フライ: フライアウト
  • ゴロ: ゴロアウト
  • 他アウト: その他のアウト（ライナーアウトなど）

2. 打撃成績の自動集計

各メンバーの打席結果を試合をまたいで合計します。

打席数 = 全結果（ヒット系 + 出塁系 + アウト系）の合計
打数 = 打席数 - 四球 - 死球
安打 = ヒット + 二塁打 + 三塁打 + 本塁打
打率 = 安打 ÷ 打数（小数3位まで、打数0なら .---）
打点 = 各打席で「打点」として入力した値の合計
注意: エラー出塁・他セーフは打数に含まれますが、安打にはなりません（打率が下がります）。

3. 投手成績の自動集計

守備の打席記録（相手チームの打席）で「投手」を指定すると、その投手の成績が自動集計されます。

登板試合数 = その投手が1打席以上投げた試合の数
奪三振 = 結果が「三振」の打席数
与四球 = 結果が「四球」の打席数
与死球 = 結果が「死球」の打席数
被安打 = 結果が「ヒット / 二塁打 / 三塁打 / 本塁打」の打席数
失策出塁 = 結果が「エラー」の打席数
失点 = 各打席で「失点」として入力した値の合計
注意: 「他アウト」「他セーフ」は投手成績には特別なカウントを付けません（その他扱い）。

4. 勝ち / 負け投手の自動判定

各試合ごとに、以下のルールで勝ち投手・負け投手を自動判定します。

条件:

  • 守備の打席記録（相手の打席）が1つ以上ある
  • 試合に勝敗がある（ourScore ≠ theirScore、同点なら判定なし）

判定:

  • その試合で「最も多く相手打者と対戦した投手」 = 決定投手
  • 当チームが勝った場合（ourScore > theirScore） → 決定投手に「勝」+1
  • 当チームが負けた場合（ourScore < theirScore） → 決定投手に「負」+1
例: あさひが18打席、田中が3打席を投げた試合。
スコアは 8-3 で勝利。
→ あさひが決定投手なので、あさひに「勝」+1
注意: PTA草野球向けの簡易ルールです。プロ野球の正式な勝利投手ルール（5回以上投げた先発が条件など）とは異なります。 手動上書きしたい場合は、相手チームの打席記録の「投手」フィールドを編集してください。

5. イニングスコア・試合結果

各イニングの得点

当チームの○回得点 = その回に記録された当チームの全打席の「打点」合計
相手チームの○回得点 = その回に記録された相手の全打席の「失点」合計

試合の合計スコア

ourScore = 当チームの全打席の打点合計
theirScore = 全相手打席の失点合計

試合結果

  • ourScore > theirScore: 勝 (Win)
  • ourScore < theirScore: 負 (Lose)
  • ourScore == theirScore: 分 (Draw)
盗塁や暴投で進塁した場合の得点は「打点」には反映されないため、自動計算とズレることがあります。 必要なら「打点」の値を手動で +1 して調整してください（緩和策）。

6. イニングの自動進行

  • 「三振 / フライ / ゴロ / 他アウト」の合計が 3に達するとアウトカウントが3/3になり、次の打席は自動で次の回に進みます
  • 「前の回」「次の回」ボタンで手動移動も可能（手動指定中は黄色のラベル表示）
  • イニング数の上限はありません。時間制限の試合は「🏁 試合終了 / スコア確定」ボタンで終了させてください

7. MVP・その他

  • MVP: 試合の登録・編集モーダルで指定した選手のみ（手動指定）。自動判定はありません
  • 試合終了 / スコア確定: 時間制限などで終了した試合は、「🏁 試合終了 / スコア確定」ボタンで明示的に確定。試合カードに「🏁 確定」バッジが付きます
  • 順位（メンバー一覧）: メンバー画面の「並び替え」を成績系（打率・打席・本塁打など）にすると、メンバーカードに順位バッジが表示されます
ルールの追加・変更を希望する場合は、代表者にご相談ください。
